Image Description Generation System

Description Generation Evaluation Demonstration

Let the computer tell what the given image is about !

What is this system and what does it do ？

Humans can summarize the content of images in sentences. But, how to make computers also have this ability has become a novel research problem.My project aims to explore such problems and propose an interactive system. This website is a part of my MSC project in the University of Manchester.

The system architecture has three layers, including a visual-language model layer that contains Encoder (Resnet 101 or Resnet152) and Decoder (LSTM) with hard attention. A server layer that implements user interaction with the model layer. A web layer that connects designed web pages (in here!).

See the navigation bar above, you can click on Resources button to find some necessary resources like pretrained model, source code, experimental data. Click on TEST THE MODEL button to upload one or more images and then the system will use pre-trained fine-tuned model which trained by me to get the description of uploaded images (may take some time). Click on Some Demos button to view the results of some test pictures being tested by the model. Test images are from the COCO and Flick30K data sets.
