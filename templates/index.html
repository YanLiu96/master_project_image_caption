What is this system and what does it do ï¼Ÿ

Humans can summarize the content of images in sentences. But, how to make computers also have this ability has become a novel research problem. My project aims to explore such problems and propose solutions. This website is a part of my MSC project. It visualizes the image description generation system developed by Yan Liu. As we known, image description generation system is a special area of artificial intelligence, which attracted much attention in recent years. It requires many related technologies of computer vision and natural language processing. In short, this system aims to describe what the uploaded image is about, which means the system is able to either map images to a candidate pool of sentence descriptions or to generate novel text descriptions for input images. See the navigation bar above, you can click on TRAIN A MODEL button to train a model. Or click on TEST THE MODEL button to upload images and then the system will use pre-trained model which trained by you or me to get the description of uploaded images.
